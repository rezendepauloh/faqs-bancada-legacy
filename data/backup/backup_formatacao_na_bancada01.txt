<h1>Passo a passo</h1>

								<p>Como utilizamos um servidor de imagem, preparamos a máquina com o básico:
								</p>

								<ol>
									<li>A estação de trabalho (computador)</li>
									<li>O cabo de força</li>
									<li>O cabo de rede</li>
									<li>O cabo do monitor (seja <samp>VGA</samp>, <samp>DisplayPort</samp>
										ou
										<samp>HDMI</samp>)
									</li>
								</ol>

								<picture>
									<source srcset="img/formatacao_remota_01.jpeg" type="image/svg+xml">
									<img src="img/formatacao_remota_01.jpeg" class="img-fluid img-thumbnail"
										width="500px" title="Computador com os plugues básicos"
										alt="Computador com os plugues básicos">
								</picture>

								<p>Dependendo da estação, ao pressionar o botão de ligar, já começa a
									pressionar o
									botão que seleciona o Menu de <samp>boot</samp>. A maioria delas é o
									<kbd>F12</kbd>.
								</p>

								<!-- Colocar uma imagem com a tela do boot menu, ou... criar uma tabela com fotos de cada boot diferente, das máquinas diferentes de nosso parque -->

								<p>Escolha a opção que utiliza a rede, como <strong>"<samp>Network
											boot</samp>"</strong> ou algo parecido</p>

								<p>Aqui é preciso aguardar a comunicação entre a placa de rede (mostrada no
									<samp>endereço MAC</samp>) se comunicar com o servidor <samp>PXE</samp>,
									aqui
									poderá ocorrer dois casos:
								</p>

								<ol>
									<li>Caso ele esteja no <samp>PXE</samp>, ele se negará a entrar no
										<samp>boot</samp> pela rede e escolherá outra opção de
										<samp>boot</samp>,
										conforme a imagem abaixo:
									</li>

									<picture>
										<source srcset="img/formatacao_remota_pxe_block.jpeg" type="image/svg+xml">
										<img src="img/formatacao_remota_pxe_block.jpeg" class="img-fluid img-thumbnail"
											width="800px" title="PXE bloqueado" alt="PXE bloqueado">
									</picture>

									<li>Nesse caso, é preciso entrar no <strong><samp>SCCM</samp></strong>
										na sua
										estação de trabalho, localizar a máquina (pode ser pelo
										<var>patrimônio</var>, <var>último usuário logado</var> ou
										<var>Endereço
											MAC</var> da placa de rede), pressionar o botão direito do
										<kbd>Mouse</kbd> e selecionar a opção <kbd>Delete</kbd>.
									</li>

									<picture>
										<source srcset="img/formatacao_remota_02.jpg" type="image/svg+xml">
										<img src="img/formatacao_remota_02.jpg" class="img-fluid img-thumbnail"
											width="1000px" title="SCCM excluindo uma estação"
											alt="SCCM excluindo uma estação">
									</picture>

									<li>E confirmar a exclusão pressionando o botão <samp>Delete</samp> da
										janela
										que se abrir.</li>

									<picture>
										<source srcset="img/formatacao_remota_pxe_confirm_delete.jpg"
											type="image/svg+xml">
										<img src="img/formatacao_remota_pxe_confirm_delete.jpg"
											class="img-fluid img-thumbnail" width="400px"
											title="SCCM excluindo uma estação 2" alt="SCCM excluindo uma estação 2">
									</picture>

									<li>Depois, reinicie a máquina que iremos formatar e reincie o processo.
										Vai
										aparecer a opção de pressionar <kbd>F12</kbd> no fim, conforme a
										imagem
										abaixo.</li>

									<picture>
										<source srcset="img/formatacao_remota_sem_pxe_f12.jpeg" type="image/svg+xml">
										<img src="img/formatacao_remota_sem_pxe_f12.jpeg"
											class="img-fluid img-thumbnail" width="800px"
											title="PXE liberando para fazer a formatação"
											alt="PXE liberando para fazer a formatação">
									</picture>

									<li>Na próxima opção, é só escolher a única opção e aguardar a
										finalização do
										<samp>boot</samp>.
									</li>

									<picture>
										<source srcset="img/formatacao_remota_ramdisk_options.jpeg"
											type="image/svg+xml">
										<img src="img/formatacao_remota_ramdisk_options.jpeg"
											class="img-fluid img-thumbnail" width="800px" title="PXE opção do ramdisk"
											alt="PXE opção do ramdisk">
									</picture>

								</ol>

								<p>A partir daqui, ele começará a carregar o <var>Image Server</var>.</p>

								<picture>
									<source srcset="img/formatacao_remota_03.jpeg" type="image/svg+xml">
									<img src="img/formatacao_remota_03.jpeg" class="img-fluid img-thumbnail"
										width="800px" title="Image Server em execução" alt="Image Server em execução">
								</picture>

								<p>A primeira janela é a tela de autenticação. É só colocar a senha
									<strong>MPMS.0000</strong>
								</p>

								<picture>
									<source srcset="img/formatacao_remota_04.jpeg" type="image/svg+xml">
									<img src="img/formatacao_remota_04.jpeg" class="img-fluid img-thumbnail"
										width="800px" title="Autenticação do Image Server"
										alt="Autenticação do Image Server">
								</picture>

								<p>A próxima tela é a escolha do sistema operacional. Aqui é preciso
									analisar o
									seguinte:</p>

								<ol>

									<picture>
										<source srcset="img/formatacao_remota_05.jpeg" type="image/svg+xml">
										<img src="img/formatacao_remota_05.jpeg" class="img-fluid img-thumbnail"
											width="800px" title="Escolha do Sistema Operacional"
											alt="Escolha do Sistema Operacional">
									</picture>

									<li>O primeiro é o Sistema Windows 10 para a <strong>PGJ</strong> e o
										<strong>GAECO</strong>. Caso a estação seja para algum desses
										prédios,
										selecionar esse.
									</li>

									<li>O segundo é o Sistema Windows 10 para as <strong>DEMAIS
											LOCALIDADES</strong>, ou seja, para as localidades além da PGJ e
										do
										GAECO (Interior e promotorias de Campo Grande).</li>

									<li>O terceiro é o Sistema Windows 10 para as
										<strong>NOTEBOOKS</strong>.
										Qualquer laptop pode usar essa opção.
									</li>
								</ol>

								<p>A próxima e última tela é o nome da estação que aparecerá posteriormente
									no
									<samp>AD</samp> e <samp>SCCM</samp>. Ela é opcional pois podemos
									colocá-la
									depois, mas geralmente já preenchemos aqui para evitar um trabalho
									posterior.
									Basta pressionar <kbd>Tab</kbd> até o botão <samp>Edit</samp> ou
									posicionar o
									ponteiro do mouse e clicar sobre ele.
								</p>

								<picture>
									<source srcset="img/formatacao_remota_06.jpeg" type="image/svg+xml">
									<img src="img/formatacao_remota_06.jpeg" class="img-fluid img-thumbnail"
										width="800px" title="Seleção do Edit" alt="Seleção do Edit">
								</picture>

								<p>Na janela que se abrirá, basta pressionar <kbd>Tab</kbd> até o formulário
									<samp>Value:</samp> e digitar o nome do computador. Após digitá-lo,
									pressione o
									botão <samp>OK</samp>
								</p>

								<picture>
									<source srcset="img/formatacao_remota_07.jpeg" type="image/svg+xml">
									<img src="img/formatacao_remota_07.jpeg" class="img-fluid img-thumbnail"
										width="800px" title="Preenchimento do nome da estação"
										alt="Preenchimento do nome da estação">
								</picture>

									tabela a
									seguir mostra as siglas de todos os prédios do MP:
